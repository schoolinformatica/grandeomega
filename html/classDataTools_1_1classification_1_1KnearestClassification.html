Detailed Description

Classification algorithm using K Nearest.

K Nearest is a classification algorithm that uses a training set in order te classify new points. To do so, K Nearest computes the distance from the new point to all the points in the training set. With a time complexity of O(m*n) where n is the length of the training set and m is the length of the set of new points, this a CPU heavy operation.
