Detailed Description

Class featuring the DB Scan algorithm.

The DB Scan algorithm is capable of clustering vectors of n dimensions. As input, it needs a radius within neighbours should be together, the minimum amount of point in a cluster and ofcourse the dataset, in this case, a IEnumerable<GenericVector>. As output it produces a Dictionary DataClusters with the key as cluster and the value the vectors belonging to the cluster.
